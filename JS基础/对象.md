# 对象

### JS中有那些内置对象
 - 数据封装类对象 
   - String、Array、Object、Boolean、Number
 - 其他对象
   - Math、Date、RegExp、Error、Function、Arguments
 - ES6 新增对象
   - Promise、Map、Set、Symbol、Proxy、Reflect
